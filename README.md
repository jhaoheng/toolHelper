- gitlab
	- 版本
		- 線上 EE : 免費
		- CE 下載版 : 免費
		- EE 下載版 : 付費
	- 優點 : 用線上 EE 版，功能全支援，免費， all free repo

	- 版本：線上版
		- 免費：最高五個用戶
		- 付費：依據用戶最高上限數量，每月付費
